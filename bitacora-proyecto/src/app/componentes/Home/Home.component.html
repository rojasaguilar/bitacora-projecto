Locmer

Descripcion del problema

Hicimos este proyecto con la intencion de crear un chat que sea mas privado que otros, ya que no pide al usuario datos personales como correo o numero de telefono para poder usar Locmer chat, solamente un nombre de usuario para identificarse en la conversacion, aunque tambien se puede acceder con un nombre de usuario anonimo para aún mas privacidad

Audiencia a la que va dirigida

Nuestro chat está orientado a aquellas personas que se preocupen por su privacidad, permitiendo interactuar solo con las personas a las que se les hayan sido compartido el código de acceso. Ademas de esto, los mensajes no se guardan en servidor, aumentando aún más la privacidad de las conversaciones
